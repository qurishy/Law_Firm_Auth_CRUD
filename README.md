# Law Firm Auth CRUD

## Project Overview
The Law Firm Auth CRUD project is a web application designed to manage authentication and authorization for a law firm. This application allows users to create, read, update, and delete user accounts and roles, streamlining the process of managing access to various parts of the system.

## Features
- **User Authentication:** Secure login and registration for users.
- **Role Management:** Create and manage user roles to control access levels.
- **CRUD Operations:** Full support for creating, reading, updating, and deleting user accounts.
- **Responsive Design:** Accessible and user-friendly interface for both desktop and mobile devices.

## Technology Stack
- **Frontend:** React.js
- **Backend:** Node.js with Express.js
- **Database:** MongoDB
- **Authentication:** JSON Web Tokens (JWT)
- **Styling:** CSS with Bootstrap

## Setup Instructions
1. **Clone the repository:**  
   ```bash
   git clone https://github.com/qurishy/Law_Firm_Auth_CRUD.git
   ```
2. **Navigate to the project directory:**  
   ```bash
   cd Law_Firm_Auth_CRUD
   ```
3. **Install dependencies:**  
   ```bash
   npm install
   ```
4. **Set up environment variables:**  
   Create a `.env` file in the root directory and add your configuration.
5. **Run the application:**  
   ```bash
   npm start
   ```

## Usage Guidelines
- **Access the application:** Open your browser and navigate to `http://localhost:3000`.
- **Register a new account** by clicking on the "Sign Up" button.
- **Log in** with your credentials and manage your profile.
